# Q2-3

核心目標：使用 Verilog 撰寫計數器，根據輸入值調整 High/Low 的比例在 Vivado 中模擬，截圖三種不同工作週期的波形

流程：建立計數器 ＞ 比較輸入值 ＞ 計算工作週期 ＞ 模擬驗證

建立計數器 ：
計數器從 0 數到 9（共 10 格）
代表一個完整的 PWM 週期

比較輸入值：
當計數器 < 輸入值 → 輸出 1（HIGH）
當計數器 ≥ 輸入值 → 輸出 0（LOW）

計算工作週期：
工作週期 = 輸入值 ÷ 10 × 100%
輸入 1 → 1/10 = 10%
輸入 5 → 5/10 = 50%
輸入 8 → 8/10 = 80%

模擬驗證：
分別輸入三種不同的值
觀察波形確認 HIGH 和 LOW 的比例正確
